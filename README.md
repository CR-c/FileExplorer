# 仿Explorer结构文件管理

+ 项目描述

	主要使用链表对树形目录文件结构进行模拟，进行新建文件/文件夹，返回根目录/上级目录，查看下级目录等操作。

+ 采用技术

	+ 链表
	+ 栈

+ 实现功能

	+ 打开目录文件夹
	+ 新建文件/文件夹
	+ 删除文件/文件夹
	+ 重命名文件/文件夹
	+ 排序所有文件
	+ 目录文件大小统计
	+ 支持返回上级目录及根目录操作

+ 开发语言

	+ **C 语言（C 99）**

+ 开发环境

	+ IDE：**CLion 2019.3.2**
	+ CMake：**CMake Version 3.15**
	+ GCC：**mingw-w64 6.0**
	+ Debugger： **MinGW-w64 GDB Version 8.1**
